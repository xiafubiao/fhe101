# FHE101

全同态加密（Fully Homomorphic Encryption，FHE）是一种革命性的密码学技术，允许对加密数据直接进行计算，无需先解密数据。

**推荐入门资料**：[The Beginner's Textbook for Fully Homomorphic Encryption](https://arxiv.org/abs/2503.05136) 是一本 600+ 页的系统教程，从零基础开始。配套 Python 代码：[fhe-textbook](https://github.com/fhetextbook/fhe-textbook)
