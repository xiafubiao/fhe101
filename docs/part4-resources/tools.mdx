
## 常用库

### C++生态
- [OpenFHE](https://github.com/openfheorg/openfhe-development) - 全功能 FHE 库
- [Microsoft SEAL](https://github.com/microsoft/SEAL) - BFV/CKKS
- [HEAAN](https://github.com/snucrypto/HEAAN) - CKKS 优化

### Rust 生态
- [TFHE-rs](https://github.com/zama-ai/tfhe-rs) - 布尔 FHE
- [FHE.rs](https://github.com/FHE-org/fhe.rs) - Rust FHE 生态

### Python 生态
- [PySEAL](https://pypi.org/project/pyseal/) - SEAL Python 绑定
- [TenSEAL](https://github.com/OpenMined/TenSEAL) - CKKS  Python 绑定
- [Concrete](https://github.com/zama-ai/concrete) - TFHE Python
