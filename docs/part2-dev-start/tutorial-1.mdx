# 案例一：隐私数据统计

## 场景

医院 A 和医院 B 想要联合计算某种疾病的平均治疗费用，但双方都不想暴露各自的患者费用数据。

本案例使用 Microsoft SEAL，官方提供以下示例代码：

## 官方 BFV 示例核心代码

来源：[1_bfv_basics.cpp](https://github.com/microsoft/SEAL/blob/main/native/examples/1_bfv_basics.cpp)

```cpp
// 1. 创建加密参数 (BFV 方案)
EncryptionParameters parms(scheme_type::bfv);
parms.set_poly_degree(4096);
parms.set_coeff_modulus(CoeffModulus::BFVDefault(4096));
parms.set_plain_modulus(PlainModulus::Batching(4096, 20));

SEALContext context(parms);

// 2. 生成密钥
KeyGenerator keygen(context);
PublicKey public_key;
keygen.create_public_key(public_key);
SecretKey secret_key = keygen.secret_key();

// 3. 创建编码器、加密器、评估器
BatchEncoder batch_encoder(context);
Encryptor encryptor(context, public_key);
Evaluator evaluator(context);
Decryptor decryptor(context, secret_key);

// 4. 编码并加密
vector<uint64_t> pod_matrix(512);
transform(pod_matrix.begin(), pod_matrix.end(), pod_matrix.begin(), [&](auto) { return uniform_int_distribution<uint64_t>(0, 25)(engine); });

Plaintext plain_matrix;
batch_encoder.encode(pod_matrix, plain_matrix);
Ciphertext encrypted_matrix;
encryptor.encrypt(encrypted_matrix, plain_matrix);

// 5. 在密文上执行加法
Ciphertext encrypted_add;
evaluator.add_inplace(encrypted_matrix, encrypted_matrix);

// 6. 解密并解码
Plaintext plain_result;
decryptor.decrypt(encrypted_matrix, plain_result);
vector<uint64_t> result_matrix;
batch_encoder.decode(result_matrix, plain_result);
```

## 关键点总结

1. **数据不泄露**：医院 A 只看到加密数据，无法得知医院 B 的具体费用
2. **计算正确**：解密后的平均值是两家医院真实数据的平均值
3. **噪声管理**：实际使用时需要预估噪声预算，选择合适的参数
