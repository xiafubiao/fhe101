|-----|------|----------|------|----------|
| **OpenFHE** | C++ | BFV, CKKS, TFHE, FHEW | 功能最全面，工业级 | 生产环境 |
| **Microsoft SEAL** | C++, C#, Python | BFV, CKKS | 微软维护，文档完善 | 学术研究 |

## 快速开始：Hello FHE

以 TFHE-rs 为例，5 分钟体验同态加密：

```rust
use tfhe::prelude::*;
use tfhe::{generate_keys, set_server_key, ConfigBuilder};

fn main() {
    // 1. 配置参数
    let config = ConfigBuilder::all_disabled()
        .enable_default_integers()
        .build();

    // 2. 生成密钥
    let (client_key, server_key) = generate_keys(config);

    // 3. 在客户端加密
    set_server_key(server_key);
    let a = FheInt32::encrypt(42, &client_key);
    let b = FheInt32::encrypt(1, &client_key);

    // 4. 云端在密文上计算（不接触明文）
    let result = &a + &b;

    // 5. 客户端解密
    let decrypted: i32 = result.decrypt(&client_key);
    println!("42 + 1 = {}", decrypted); // 输出 43
}
```
