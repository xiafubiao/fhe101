# FHE开发工具

## 主流 FHE SDK 介绍

| SDK | 语言 | 主要方案 | 特点 | 推荐场景 |
|-----|------|----------|------|----------|
| **OpenFHE** | C++ | BFV, CKKS, TFHE, FHEW | 功能最全面，工业级 | 生产环境 |
| **Microsoft SEAL** | C++, C#, Python | BFV, CKKS | 微软维护，文档完善 | 学术研究 |
| **TFHE-rs** | Rust | TFHE | 性能优秀，社区活跃 | 布尔运算、快速部署 |
| **HEAAN** | C++ | CKKS | 适合浮点运算 | 机器学习场景 |
| **PySEAL** | Python | BFV, CKKS | 易上手 | 快速原型验证 |

## 快速开始：Hello FHE

以 TFHE-rs 为例，5 分钟体验同态加密：

```rust
use tfhe::{ConfigBuilder, generate_keys, set_server_key, FheUint8};
use tfhe::prelude::*;

fn main() {
    let config = ConfigBuilder::default().build();

    // 客户端
    let (client_key, server_key) = generate_keys(config);

    let clear_a = 27u8;
    let clear_b = 128u8;

    let a = FheUint8::encrypt(clear_a, &client_key);
    let b = FheUint8::encrypt(clear_b, &client_key);

    // 服务端
    set_server_key(server_key);
    let result = a + b;

    // 客户端
    let decrypted_result: u8 = result.decrypt(&client_key);

    let clear_result = clear_a + clear_b;

    assert_eq!(decrypted_result, clear_result);
}
```
