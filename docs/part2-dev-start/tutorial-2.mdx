# 案例二：隐私 Swap

## 场景：为什么需要隐私 Swap？

在传统 AMM（自动做市商，如 Uniswap）中，所有交易都暴露在链上：

- 交易者提交的订单金额是公开的
- 攻击者可以观察到_pending_交易
- 通过_front-running_（抢跑攻击）操纵价格获利

**隐私 Swap** 可以解决这一问题：交易者加密自己的订单金额，矿工/验证者在密文上执行计算，全程不知道实际交易量。

## AMM 工作原理

AMM 是去中心化交易所的核心机制：

```
池子（Pool）包含两种代币：
- ETH 池：100 ETH
- NU 池：1,000 NU

交易公式（恒定乘积公式）：
x * y = k（其中 k 是常数）

交换比例由池子储量决定：
ETH 数量 × NU 数量 = 常数
```

## FHE 如何实现隐私 Swap

来源：[Sunscreen BFV Documentation - Private token swap](https://bfv-docs.sunscreen.tech/fhe/fhe_programs/example.html)

Alice 想用 NU 代币兑换 ETH，但不想让别人知道她兑换了多少。

### 核心流程

```
1. Alice 用公钥加密自己的 NU 数量（如 20 NU）
2. Alice 把加密后的订单发送到链上
3. 矿工在密文上执行 Swap 计算（不知道实际数量）
4. 矿工返回加密后的 ETH 数量
5. Alice 用私钥解密，得到实际收到的 ETH
```

### FHE 程序：swap_nu

这是核心的 FHE 程序，使用 BFV 方案：

```rust
#[fhe_program(scheme = "bfv")]
/// 这个程序计算 NU 换 ETH 的汇率
fn swap_nu(
    nu_tokens_to_trade: Cipher<Rational>,
) -> Cipher<Rational> {
    let total_eth = 100.0;
    let total_nu = 1_000.0;

    // 恒定乘积公式
    -(total_eth * total_nu / (total_nu + nu_tokens_to_trade) - total_eth)
}
```

**关键点**：
- 输入是加密的 NU 数量（`Cipher<Rational>`）
- 输出是加密的 ETH 数量
- 池子储量（`total_eth`、`total_nu`）是公开的
- 使用 `Rational` 类型因为涉及密文上的除法

### Alice（交易者）

Alice 需要：
1. 生成公钥/私钥对
2. 加密她的交易数量
3. 最后解密查看收到的 ETH

```rust
struct Alice {
    pub public_key: PublicKey,
    private_key: PrivateKey,
    runtime: FheRuntime,
}

impl Alice {
    // 创建加密交易
    pub fn create_transaction(&self, amount: f64) -> Result<Ciphertext, Error> {
        Ok(self.runtime
            .encrypt(Rational::try_from(amount)?, &self.public_key)?
        )
    }

    // 解密查看收到的 ETH
    pub fn check_received_eth(&self, received_eth: Ciphertext) -> Result<(), Error> {
        let received_eth: Rational = self
            .runtime
            .decrypt(&received_eth, &self.private_key)?;
        println!("Alice received {}ETH", received_eth);
        Ok(())
    }
}
```

### 矿工/验证者

矿工执行 FHE 程序，但不知道实际交易量：

```rust
struct Miner {
    pub compiled_swap_nu: CompiledFheProgram,
    runtime: FheRuntime,
}

impl Miner {
    pub fn run_contract(
        &self,
        nu_tokens_to_trade: Ciphertext,
        public_key: &PublicKey,
    ) -> Result<Ciphertext, Error> {
        // 在密文上运行 Swap 计算
        let results = self.runtime.run(
            &self.compiled_swap_nu,
            vec![nu_tokens_to_trade],
            public_key,
        )?;
        Ok(results[0].clone())
    }
}
```

### 完整交互流程

```rust
fn main() -> Result<(), Error> {
    // 1. 矿工设置 Swap 合约
    let miner = Miner::setup()?;

    // 2. Alice 设置（使用矿工公布的参数）
    let alice = Alice::setup(&miner.compiled_swap_nu.metadata.params)?;

    // 3. Alice 加密她的订单（20 NU）
    let transaction = alice.create_transaction(20.0)?;

    // 4. 矿工在密文上计算应得 ETH
    let encrypted_received_eth =
        miner.run_contract(transaction, &alice.public_key)?;

    // 5. Alice 解密查看结果
    alice.check_received_eth(encrypted_received_eth)?;

    Ok(())
}
```
