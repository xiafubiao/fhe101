# FHE101

Fully Homomorphic Encryption (FHE) is a revolutionary cryptographic technique that allows computations to be performed directly on encrypted data without requiring decryption first.

**Recommended Getting Started Material**: [The Beginner's Textbook for Fully Homomorphic Encryption](https://arxiv.org/abs/2503.05136) is a 600+ page systematic tutorial starting from zero basics. Companion Python code: [fhe-textbook](https://github.com/fhetextbook/fhe-textbook)
