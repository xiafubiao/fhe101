## Common Libraries

### C++ Ecosystem
- [OpenFHE](https://github.com/openfheorg/openfhe-development) - Full-featured FHE library
- [Microsoft SEAL](https://github.com/microsoft/SEAL) - BFV/CKKS
- [HEAAN](https://github.com/snucrypto/HEAAN) - CKKS optimized

### Rust Ecosystem
- [TFHE-rs](https://github.com/zama-ai/tfhe-rs) - Boolean FHE
- [FHE.rs](https://github.com/FHE-org/fhe.rs) - Rust FHE ecosystem

### Python Ecosystem
- [PySEAL](https://pypi.org/project/pyseal/) - SEAL Python bindings
- [TenSEAL](https://github.com/OpenMined/TenSEAL) - CKKS Python bindings
- [Concrete](https://github.com/zama-ai/concrete) - TFHE Python
