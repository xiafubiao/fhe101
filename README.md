# FHE101

A developer's getting started guide to Fully Homomorphic Encryption (FHE).

## Local Preview

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Start the development server
npm start
```

Visit http://localhost:3000 to view the documentation.
